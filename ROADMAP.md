# 🛣️ ROADMAP — VLX‑16S

Cette roadmap décrit l’évolution prévue du projet du stade minimal (v1.0.0) à la version avancée (v5.0.0).

---

## 🎯 Objectif global
Créer un simulateur ECU tactique complet, modulaire, réaliste, inspiré des calculateurs militaires modernes.

---

## 🟩 Phase 1 — v1.x.x : Base du simulateur
- [x] Modèle moteur minimal (RPM / Load / Temp)
- [x] Injection & allumage basiques
- [x] Structure du projet
- [x] Logo + identité visuelle
- [ ] Documentation technique initiale

---

## 🟩 Phase 2 — v2.x.x : Modes & stabilité
- [x] Modes Civil / Tactique / Dégradé
- [x] Tests unitaires
- [ ] Gestion capteurs étendue
- [ ] Courbes d’avance plus réalistes

---

## 🟩 Phase 3 — v3.x.x : Interface & API
- [x] Interface Python (FastAPI)
- [x] Visualisation temps réel
- [ ] Dashboard tactique complet
- [ ] Export logs moteur

---

## 🟩 Phase 4 — v4.x.x : Diesel & CAN-Bus
- [x] Profil moteur Diesel 2.2L (type VT4)
- [x] CAN-bus virtuel
- [ ] Messages ECU standardisés
- [ ] Mode Dégradé+ avancé

---

## 🟩 Phase 5 — v5.x.x : Simulation haute-fidélité
- [x] Cycle moteur 4T complet
- [x] Gestion AFR / knock / lambda
- [x] Système de plugins ECU
- [ ] Mode Stealth (signature thermique réduite)
- [ ] Optimisation CPU/GPU

---

## 🚀 Vision long terme
- Simulation multi-moteurs  
- Interface 3D cockpit tactique  
- Intégration IA pour auto‑tuning
