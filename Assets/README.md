# 📦 Dossier assets/

Ce dossier contient toutes les ressources visuelles du projet **VLX‑16S**.

## 📁 Contenu
- Logos (SVG, versions statiques et animées)
- Éléments graphiques du thème tactique
- Icônes, motifs et textures
- Visuels utilisés dans la documentation et l’interface

## 🎨 Rôle du dossier
Centraliser tous les fichiers graphiques afin de :
- garder une structure propre et organisée,
- faciliter la maintenance du thème visuel,
- permettre la réutilisation des éléments dans l’ensemble du projet.

## 🛠️ Bonnes pratiques
- Utiliser des noms de fichiers explicites  
- Conserver les versions vectorielles (SVG) en priorité  
- Éviter les doublons  
- Classer par sous‑dossiers si nécessaire (ex : `logo/`, `icons/`, `hud/`)
